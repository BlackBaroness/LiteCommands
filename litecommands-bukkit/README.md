### Bukkit Extension Dependencies
Maven
```xml
<dependency>
    <groupId>dev.rollczi.litecommands</groupId>
    <artifactId>bukkit</artifactId>
    <version>1.4.3</version>
</dependency>
```
Gradle
```groovy
implementation 'dev.rollczi.litecommands:bukkit:1.4.3'
```
