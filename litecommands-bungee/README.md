### Bungeecord Extension Dependencies
Maven
```xml
<dependency>
    <groupId>dev.rollczi.litecommands</groupId>
    <artifactId>bungee</artifactId>
    <version>1.8.1</version>
</dependency>
```
Gradle
```groovy
implementation 'dev.rollczi.litecommands:bungee:1.8.1'
```
